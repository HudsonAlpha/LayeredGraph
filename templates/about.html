About PyxisMap:

PyxisMap is a collection of tools based on navigating complex biological networks using graph traversal algorithms. The main PyxisMap tool is built by combining phenotype relationships from the Human Phenotype Ontology (HPO) with phenotype-to-gene relationships from the HPO and PubTator. This enables researchers to select relevant phenotypes and retrieve a ranking of genes most related to that set of phenotypes. Additionally, the National Center for Biomedical Ontology (NCBO) Annotator is connected, allowing for automatic parsing of plain text (such as a clinical synopsis) to HPO terms.

PyxisMap also includes a Protein-Protein Interaction (PPI) network. This network is built from the ConsensusPathDB protein-protein interactions. This enables researchers to select relevant genes and identify connections between their proteins. This tool includes a visual representation of the top hits from the PPI network.
